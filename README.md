# 🏪 Tiendas 3E - Sistema de Caducidades e Inventario

Proyecto desarrollado como parte del portafolio académico en la **Politécnica**, organizado por Emilio y Oscar.


El objetivo principal es **gestionar inventarios de una tiendita real**, controlando productos, fechas de caducidad y disponibilidad, mediante tecnologías web ligeras y accesibles.

---

## 📌 Objetivo
- Crear un sistema sencillo y funcional para visualizar y administrar inventarios.
- Controlar **caducidades** de productos en tiempo real.
- Ofrecer una interfaz clara con **HTML + CSS/SCSS + JavaScript**.
- Utilizar **JSON** como base de datos ligera para productos y fechas.

---

## 📂 Estructura del Repositorio
- `index.html` → Página principal del sistema.
- `app.js` / `script.js` → Lógica en JavaScript para cargar inventarios y manejar caducidades.
- `inventario.json` → Datos de productos (nombre, cantidad, precio).
- `inventario_caducidades.json` → Fechas de caducidad asociadas a cada producto.
- `scss.html` → Estilos y pruebas de diseño.
- `.github/workflows/` → Configuración de despliegue automático.
- `README.md` → Este documento.

---

## 🛠️ Tecnologías
- **HTML5** → Interfaz principal.
- **CSS/SCSS** → Estilos responsivos y personalización visual.
- **JavaScript (ES6)** → Funcionalidad dinámica, carga de datos y validaciones.
- **JSON** → Almacenamiento ligero de inventario y caducidades.

---

## 🚀 Despliegue
